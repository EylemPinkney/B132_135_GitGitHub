*****GIT DAY1*****


Git init - We use git init to take my project in my local computer to the version system. 

1 - Working space

git add . --> We use this command to send our work to the Staging Area

2 - Staging area

git commit -m"message" --> we send our work to Commit Store

3 - Commit store

git log --oneline --> We see the commits we have done so far

git show first7charactersofthehashcode


***Main Commands

git init
git add .
git commit -m"Version Message"


***Support Commands

git status
git log  
git log --oneline
git show (hash_codu, first 7 digits)
